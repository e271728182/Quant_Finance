This simple example is to show how to use the various data structure and Quantlib functions to manage output and apply the correct discounting or whatever metric of your choice.
The idea is that you would query your database with the right relational data model in 3rd normal form and then apply Quantlib's mathemical finance methods on your output and send it back to a database.

Basic way of using data structs¶

Essentially because you need a lot inputs to define time,interest rates and a given model's properties those are encapsulated in the equivalent of a c++ struct which is essentially a class with no methods
those structs are passed to wrapper functions or direct Quantlib methods to do various things

Data Model¶

The following below is a skeleton of a data model.

essentially you have some kind of model which has specific characterstics such as the maker (GGY, InHouse,CatModel etc) and/or the cie who owns it.
this model produces outputs which also have specific characrteristics such as a valuation date and a time format.

Performing operations on these outputs produces calculated results which are linked to the model. Obviously some infos are missing and it should be more elaborate but it is the general idea is something like below:

Base.metadata.drop_all(engine)

class Model(Base):
    __tablename__ = 'Model'
    
    id = Column(Integer, primary_key=True)
    cie_id=Column(Integer,nullable=False)
    calculated_metrics = relationship("CalculatedResults", backref='model',collection_class=list)
    outputs=relationship("Output", backref='model',collection_class=list)
    # relationship tool allows to make joins if necessary to pick up objects which have the required relationship
    
class Output(Base):
    __tablename__ = 'Output'
    id = Column(Integer, primary_key=True)
    model_key=Column(Integer,ForeignKey('Model.id'))
    valuation_date=Column(Date,nullable=True)
    

class CalculatedResults(Base):
    __tablename__ = 'Calcresults'
    id = Column(Integer, primary_key=True)
    model_key=Column(Integer,ForeignKey('Model.id'))
    output_key=Column(Integer,ForeignKey('Output.id'))
    metric=Column(String,nullable=False)
    value=Column(Float,nullable=True)
     
Base.metadata.create_all(engine)
print(engine.table_names())
